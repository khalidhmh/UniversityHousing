## ğŸ” Security Zones

### Zone 1: Frontend (React)
```
Responsibilities:
- Collect user input (email, password)
- Form validation
- Show UI feedback (loading, errors)
- Call auth API

NOT responsible for:
- Password verification
- Database access
- Account permissions
- Session validation
```

### Zone 2: IPC Bridge
```
Responsibilities:
- Route requests to Electron main process
- Validate channel names
- Enforce whitelisted methods only

NOT responsible for:
- Business logic
- Database queries
- Password hashing
```

### Zone 3: Electron Main Process
```
Responsibilities:
- Receive IPC requests
- Validate input
- Query database
- Implement business logic
- Return validated responses

NOT responsible for:
- UI rendering
- Frontend validation
- Session persistence
```

### Zone 4: Database
```
Responsibilities:
- Store user data
- Provide data queries
- Maintain data integrity
- Index for performance

NOT responsible for:
- Business logic
- Authentication
- Session management
```
